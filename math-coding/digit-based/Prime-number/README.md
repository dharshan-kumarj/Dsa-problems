Write a program to check whether a given number N is prime or not. A number is prime if it has exactly two factors: 1 and itself.
Input Format

A single integer N
Constraints

1 <= N <= 10^5
Output Format

Print "Prime" if the number is prime
Print "Not Prime" otherwise
Sample Input 0

10
Sample Output 0

Not Prime
Sample Input 1

2
Sample Output 1

Prime
