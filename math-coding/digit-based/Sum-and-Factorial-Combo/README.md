Write a program that takes a number N and prints:
The sum of the first N natural numbers.
The factorial of N.
Input Format

A single integer N
Constraints

1 <= N <= 20

Output Format

First line: Sum of first N natural numbers
Second line: Factorial of N
Sample Input 0

5
Sample Output 0

15
120
Sample Input 1

1
Sample Output 1

1
1
