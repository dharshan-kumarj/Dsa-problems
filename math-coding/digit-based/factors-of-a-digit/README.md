Write a program to print all the factors of a given number N. A factor is a number that divides N without leaving a remainder.
Input Format

A single integer N
Constraints

1 <= N <= 10^4
Output Format

Print all the factors of N in ascending order, separated by a space.
Sample Input 0

12
Sample Output 0

1 2 3 4 6 12
Sample Input 1

7
Sample Output 1

1 7
