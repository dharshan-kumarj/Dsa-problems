You are given two positive integers base and exponent. Write a program to calculate base raised to the power exponent (base^exponent) using loop-based multiplication (not using Math.pow() or any built-in power functions).
Input Format

Two space-separated integers: base exponent
Constraints

1 ≤ base ≤ 100
0 ≤ exponent ≤ 10
Output Format

Print the result
Sample Input 0

2 3
Sample Output 0

8
Sample Input 1

2 0
Sample Output 1

1
