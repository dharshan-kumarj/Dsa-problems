A perfect number is a positive integer that is equal to the sum of its proper divisors (excluding itself).
Write a program to check if a given number N is a perfect number.
Input Format

A single integer N
Constraints

1 <= N <= 10^5
Output Format

Print "Perfect Number" if N is perfect
Otherwise, print "Not a Perfect Number"
Sample Input 0

6
Sample Output 0

Perfect Number
Sample Input 1

12
Sample Output 1

Not a Perfect Number
