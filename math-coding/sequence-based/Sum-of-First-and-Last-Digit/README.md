Given an integer N, write a program to find the sum of the first and last digit in the absolute value of N.
Input Format

A single integer N
Constraints

-10^9 <= N <= 10^9
Output Format

Print a single integer — the sum of the first and last digit.
Sample Input 0

12345
Sample Output 0

6
Explanation 0

First digit = 1, Last digit = 5, sum = 6

Sample Input 1

-907
Sample Output 1

16
Explanation 1

First digit = 9, Last digit = 7, sum = 16
