Given an integer N, write a program to check if the number is a palindrome.
A palindrome number reads the same backward as forward.
If N is a negative number, it cannot be a palindrome. In such cases, print false.
Input Format

Given an integer N, write a program to check if the number is a palindrome.
A palindrome number reads the same backward as forward.
If N is a negative number, it cannot be a palindrome. In such cases, print false.
Constraints

-10^9 ≤ N ≤ 10^9
Output Format

Print true if the number is a palindrome, otherwise print false.
Sample Input 0

121
Sample Output 0

true
Sample Input 1

-121
Sample Output 1

false
Sample Input 2

100
Sample Output 2

false
