Given an integer N, write a program to print the reverse of the number.
If N is negative, the reversed number should also be negative.
Input Format

A single integer N
Constraints

-10^9 <= N <= 10^9
Output Format

Print the reversed integer preserving the sign.
Sample Input 0

1234
Sample Output 0

4321
Sample Input 1

-907
Sample Output 1

-709
