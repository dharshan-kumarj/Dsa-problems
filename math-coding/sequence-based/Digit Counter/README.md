Given a positive integer N, write a program to count the number of digits in N.
Input Format

A single integer N
Constraints

-10^9 <= N <= 10^9
Output Format

Print the count of digits in the absolute value of N.
Sample Input 0

-12345
Sample Output 0

5
Sample Input 1

756
Sample Output 1

3
Sample Input 2

0
Sample Output 2

1
