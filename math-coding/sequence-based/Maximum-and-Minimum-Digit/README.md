Given an integer N, write a program to find the maximum and minimum digit in the absolute value of N.

Input Format

A single integer N
Constraints

-10^9 <= N <= 10^9
Output Format

Print two space-separated integers:
max_digit min_digit
Sample Input 0

12345
Sample Output 0

5 1
Sample Input 1

-907
Sample Output 1

9 0
Sample Input 2

4873
Sample Output 2

8 3
