Given an integer N, write a program to calculate:
The sum of all odd digits in the absolute value of N
The sum of all even digits in the absolute value of N
Input Format

A single integer N
Constraints

-10^9 <= N <= 10^9
Output Format

Print two space-separated integers:
sum_of_odd_digits sum_of_even_digits
Sample Input 0

12345
Sample Output 0

9 6
Sample Input 1

-407
Sample Output 1

7 4
