Given an integer N (can be positive, negative, or zero), write a program to calculate the sum of the digits of the absolute value of N.
Input Format

A single integer N
Constraints

-10^9 <= N <= 10^9
Output Format

Print the sum of digits of the absolute value of N.
Sample Input 0

12345
Sample Output 0

15
Sample Input 1

-407
Sample Output 1

11
